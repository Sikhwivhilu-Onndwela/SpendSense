# 💸 SpendSense
Smart Spending for South African Students
🌍 **Live Website:** https://sikhwivhiluo.pythonanywhere.com/

## 📌About The Project

My name is Sikhwivhilu Onndwela and I am a student from CUT. I built SpendSense because I personally experienced running out of money before month end and I wanted to create something that could help me and other students like me.

## 🚨 The Problem

-73% of SA students run out of money before month end
-R450 is wastedmonthly on unplanned spending
-1 in 3 students skip meals due to poor budgetting-notlack of funds

Most financial apps are too complex, too expensive, or not built for the South African student reality.
**SpendSense changes that.**

## ✨ Features

| Feature  | Description |
|----------|-------------|
| 📊 Budget Tracker | Enter you allowance and expenses, get a full visual breakdown |
|💡Money Tips | SA-specific saving advice on food, transport, airtime and savings |
| 🎯 Spending Scorecard|Get colour coded feedback-green, orange or red |
| 🇿🇦 Built for SA| Every tip tailored to the South African student reality |
| 🔓 Always Free| No sign up, no downloads, no hidden costs |
| 🔒 No Data Stored | Your financial info stays private - nothing is saved.|

## 🚀 How to Run Locally

**Step 1-Clone the repo**
```bash
git clone https://github.com/Sikhwivhilu-Onndwela/SpendSense.git
cd SpendSense
```

**Step 2-Create a virtual environment**
```bash
python -m venv venv
venv\Scripts\activate
```

**Step 3 - Install dependencies**
```bash
pip install -r requirements.txt
```

**Step 4- Run the Server**
```bash
python manage.py runserver
```

**Step 5 -Open in browser**
http://127.0.0.1:8000
